еңЁLinux x64зҺҜеўғдёӢgccзј–иҜ‘32дҪҚзЁӢеәҸеҸҠLibrary

2014-11-07 C/C++ GCC, Linux 0 иҜ„и®ә еӯ—ж•°з»ҹи®Ў: 427(еӯ—) йҳ…иҜ»ж—¶й•ҝ: 1(еҲҶ)

дёҖзӣҙйғҪеңЁx64жңҚеҠЎеҷЁдёҠејҖеҸ‘е’ҢйғЁзҪІ, еҒ¶е°”йҒҮеҲ°дәҶеҮ еҸ°x86жңҚеҠЎеҷЁ, иҠұдәҶдёҖеӨ©зҡ„ж—¶й—ҙеҺ»зј–иҜ‘еҮәеҗ„з§Қx86зүҲжң¬зҡ„еә“, еңЁзҪ‘дёҠжҗңдәҶдёӢеӨ§еӨҡж•°еӣһеӨҚйғҪеҫҲз®ҖеҚ•, дҪҶжҳҜе®һйҷ…дёҠеҜ№дәҺдёҖдёӘеј•з”Ёеә“жҜ”иҫғеӨҡзҡ„зЁӢеәҸе°ұйә»зғҰеӨҡдәҶ, иҝҷйҮҢи®°еҪ•дёӢжҲ‘зҡ„иҝҮзЁӢ, з”Ёзҡ„жҳҜgcc.

  1. йҰ–е…Ҳ, зЁӢеәҸзҡ„зј–иҜ‘иҰҒеҠ -m32йҖүйЎ№, е…·дҪ“жҖҺд№ҲеҠ , еҠ еңЁе“ӘйҮҢж №жҚ®йЎ№зӣ®иҮӘиә«иҖҢе®ҡ.иҰҒжіЁж„Ҹзҡ„жҳҜ, дёҚд»…CXXFLAGS (or CFLAGS)иҰҒеҠ , жңҖдё»иҰҒзҡ„жҳҜй“ҫжҺҘйҖүйЎ№д№ҹиҰҒжңү,еҗҰеҲҷg++ вҖ“WL д№Ӣзұ»зҡ„й“ҫжҺҘе‘Ҫд»ӨеңЁжІЎжңү-m32дёӢдјҡеҜ№жүҫеҲ°зҡ„32дҪҚLibжҸҗзӨә:вҖқSkip incompatible file type вҖҰ.вҖқ.
